# Housing Price Analysis

## Overview
This project analyzes housing data to identify key factors influencing property prices using multiple linear regression.

## Tools Used
- R (tidyverse, ggplot2)
- Data visualization techniques
- Regression modeling

## Key Insights
- Property size and location significantly impact price
- Regression model helps predict housing prices based on features

## Files
- Assignment 5.Rmd → analysis code
- housing.csv → dataset
- PDF report → final results and explanation
